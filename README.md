# openfl-shaders
Shaders for your openfl Sprites

## Blend Modes

* Overlay `sprite.shader = new openfl.shaders.blendMode.Overlay(Assets.getBitmapData("foreground.png"));`

## Effects

* Deuteranopia (color blindness)
* DotScreen
* Film Shader
* GrayScale
